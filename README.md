# Annual-Report-IT-Research---Right-to-Information-Commission
A strategic analysis of institutional transparency in Ghana. This repository features data-driven insights from the 2024 RTI Annual Report of the Right to Information Commission and research on the critical role of IT infrastructure in scaling public access to information. 

In addition to analysing compliance metrics, this project includes specialised research into the interdependence between IT infrastructure and freedom of information. It explores how digitised workflows and robust technical frameworks are essential for scaling the public's access to information across Ghana's public institutions.

## Repository Structure
- The markdown file of snippets from the 2024 annual reports.

- The markdown file of the research into the **interdependence between IT infrastructure and access of information.**
